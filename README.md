# AL HORRA FOODS

Static showcase website for AL HORRA FOODS, a Moroccan meat processing and charcuterie company. The site is built with plain HTML, CSS, and JavaScript.

## Project Structure

```text
.
├── index.html
├── styles.css
├── script.js
└── assets/
    └── logo.png
```

## Features

- Responsive landing page with sections for Accueil, A propos, Produits, Qualite & Certifications, and Contact.
- Fixed navigation with mobile menu.
- Hero image with parallax-style background movement.
- Loading animation and scroll reveal effects.
- Animated metric counters.
- Contact form interaction placeholder.

## Run Locally

Because this is a static site, you can open `index.html` directly in a browser.

For a local server preview, run one of these commands from the project folder:

```bash
python -m http.server 8000
```

or:

```bash
npx serve .
```

Then open:

```text
http://localhost:8000
```

## Edit Content

- Main page content: `index.html`
- Visual styles and responsive layout: `styles.css`
- Menu, loader, parallax, reveal animations, counters, and form behavior: `script.js`
- Logo asset: `assets/logo.png`

## Notes

The page uses remote Google Fonts and Unsplash images, so an internet connection is recommended for the full visual experience.
